## Grocery shop

Simple example app using Nextjs and Twind (JIT Tailwind lib).

Creates a gorcery shop with some products, cart, favorites and orders history.

## Live

Use the live version at: https://next-groceries-gamma.vercel.app/

**Install**

```
yarn
```

**Start development**

```
yarn run dev
```
